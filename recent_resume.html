Projects

Connect Four AI

I have been developing an agent to play the game of Connect Four in order to practice using machine learning algorithms. The algorithm I am currently using is a double dueling deep Q-learning network that uses an experience replay buffer and a gradient accumulator. I also use an evolutionary algorithm in order to fine-tune the hyperparameters. I developed this algorithm in Python using the TensorFlow library and have made the code available on my GitHub account which can be found above.
